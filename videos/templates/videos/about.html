Welcome to WebDeo


Note: This is a Trail Version of WebDeo.


Lets see what WebDeo can do :
  • Add videos to WebDeo so other people can see it.
  • You can also manage your videos like --> Update, Delete.
  • Comment and Like the videos.
  • Create your own account, So you can post videos on WebDeo. To create a account you just need to fill this things --> username, first name, last name, email, password, profile picture(Optional), date of birth(Optional), bio(Optional).
  • View Others Profile (You also have your own profile).
  • Search for videos.
  • And Many More...

Problem(Sorry for this Problem!): Whenever you Like or Comment or Delete a Comment then the Web Page will Reload.

The name of this Web App is WebDeo because:
Web = Web(Internet) and Deo = Last three words of Video.
